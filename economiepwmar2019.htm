Pincode 4vwo - Markt: Vraag en aanbod Hoofdstuk 1 & 2

Elasticiteit

E = Δq/Δp
Ek = Δqa/Δpb
Ei = Δq/y

E < -1 : Elastische vraag
-1 < E < 0 : Inelastische vraag
E = 0 : Volkomen inelastische vraag

Ek < 0 : Complementair goed
Ek > 0 : Substituut

0 < Ei < 1 : Normaal goed
Ei > 1 : Luxe goed
Ei < 0 : Inferieur goed

Leer deze goed!

Rekenen met markt en aanbod

TK = TVK + TCK
TO = p x q
TW = TO - TK
GTK = TK/q
GTK = GVK + GCK
GVK = TVK/q
GCK = TCK/q
GO = To/q
MK = ΔTK/Δq
MK = ΔTVK/Δq
MO = ΔTO/Δq

De eerste afgeleidene

MK = TK' = aq3 + bq2 + q
MO = TO' = aq3 + bq2 + q

Bijvoorbeeld:
TK = q3 + 2q2 + 5q + 1000
MK = TK' = 3q2 + 4q + 5
Omdat:
q3 + 2q2 + 5q
q3 = 1q3
(1x3)q3-1 = 3q2
(2x2)q2-1 = 4q1 = 4q
(5x1)q1-1 = 5q0 = 5

Dit is lastig! Probeer een paar oefensommen te maken of kijk in je boek voor verdere uitleg!

Break-even punt

BEP → TO = TK
BEP → GO = GTK
BEP → MO = MK
